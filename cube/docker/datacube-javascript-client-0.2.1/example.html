DataCube JavaScript SDK 예제

브라우저에서 바로 API를 확인하고, 3D Tiles/LAS/TIF/Cube/SHP/JSON 업로드부터 생성 job 제출, 레이어 확인, 뷰어 테스트까지

한 화면에서 처리할 수 있도록 구성한 한국어 예제입니다.

연결 설정

서버 주소와 API Key를 입력한 뒤 연결 상태를 먼저 확인하세요.

연결 전 
서버 주소와 API Key를 입력한 뒤 "연결 확인"을 눌러주세요.

레이어 / 작업 조회

레이어 병합
레이어 목록, 타일셋 목록, 생성 작업 목록 결과가 여기에 표시됩니다.

데이터 생성

3D Tiles 입력 업로드

일반 생성은 GLB/B3DM/zip을 지원합니다. Voxelizer 생성은 `tileset.json + 하위 content 파일 전체` 또는 동일 구조의 zip이 필요합니다.
파일 드래그 앤 드롭도 지원합니다.

아직 업로드된 3D Tiles 파일이 없습니다.
파라미터 설정
B3DM batch table 속성이 업로드 시 자동 감지됩니다.

LAS / LAZ 업로드

점군 파일 `.las`, `.laz`를 선택하세요. 업로드 결과의 `detected_crs`는 자동으로 표시됩니다.

아직 업로드된 LAS 파일이 없습니다.
파라미터 설정

TIF 업로드

`.tif`, `.tiff` 여러 파일 업로드를 지원합니다. 업로드 후 `input_dir`와 `input_files`를 함께 사용합니다.

아직 업로드된 TIF 파일이 없습니다.
파라미터 설정

NetCDF / GRIB2 업로드

`.gb2`, `.grib2`, `.grb`, `.nc`, `.nc4`, `.cdf` 또는 `.zip`을 업로드합니다.
시계열 배치는 폴더 구조나 zip 내부 경로를 유지합니다.

아직 업로드된 NetCDF/GRIB2 파일이 없습니다.
생성 모드
배치 모드는 업로드된 `upload_dir` 전체를 처리합니다. 동일 기압 U/V 파일쌍 또는 스칼라 시계열 파일이 같은 업로드 폴더 아래에 있어야 합니다.
파라미터 설정

Cube 결과 업로드

`.bdc`, `.adc`, `voxelset.json` 파일이 들어있는 폴더 전체 또는 이를 묶은 `.zip`을 올리세요.

아직 업로드된 Cube 파일이 없습니다.
파라미터 설정

SHP 업로드

`.shp/.shx/.dbf/.prj` 세트 또는 `.zip`을 업로드합니다. 여러 SHP 세트는 Application Datacube로 제출합니다.

아직 업로드된 SHP 파일이 없습니다.
DEM은 선택 사항입니다.
파라미터 설정
고급 설정

좌표 CSV 업로드

필수: longitude, latitude, 값 컬럼 1개 이상. 선택: station_id, station_name, datetime, elevation.

아직 업로드된 좌표 CSV 파일이 없습니다.

행정구역 GeoJSON 업로드

시도/시군구/읍면동 경계 `.geojson`, `.json` 파일을 업로드합니다.

아직 업로드된 GeoJSON 파일이 없습니다.

통계 CSV 업로드

행정구역 코드와 통계값을 포함한 UTF-8 CSV 파일을 업로드합니다.

아직 업로드된 통계 CSV 파일이 없습니다.
